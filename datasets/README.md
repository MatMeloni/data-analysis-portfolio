# Datasets

Esta pasta contém os conjuntos de dados utilizados nos projetos de análise.

## Conjuntos de Dados Disponíveis

### Dados Acadêmicos
- **Titanic**: Dataset clássico com informações sobre passageiros do Titanic e sua sobrevivência
- **Hotel**: Conjunto de dados com preços, avaliações e localização de hotéis em diversas cidades
- **Empresa**: Dados de funcionários incluindo salário, educação, tempo de serviço e região

### Dados Financeiros
- **Tech Companies**: Dados sobre as principais empresas de tecnologia, incluindo capitalização de mercado, setor e país

## Organização

- Cada conjunto de dados deve estar em sua própria subpasta
- Incluir um arquivo README em cada subpasta explicando a origem dos dados e sua estrutura
- Preferir formatos como CSV, JSON, ou Excel para facilitar o compartilhamento

## Dados Sensíveis

- Não armazenar dados confidenciais ou pessoais
- Se necessário, utilize técnicas de anonimização
- Considere incluir versões de amostra de grandes conjuntos de dados
